# Technical Handoff Package

This directory contains the necessary context for OpenCode to take over the debugging of the "Black Screen" issue in the Argos Portfolio Tracker.

## Contents

- **[SYSTEM_OVERVIEW.md](./SYSTEM_OVERVIEW.md)**: High-level architecture, stack details, and key dependencies.
- **[BUG_REPORT.md](./BUG_REPORT.md)**: Detailed analysis of the "Black Screen" bug, including symptoms, error logs, and hypotheses.
- **[REPRODUCTION_GUIDE.md](./REPRODUCTION_GUIDE.md)**: Instructions on how to set up the project and reproduce the bug.

## Instructions for OpenCode

1.  Start by reading the **BUG_REPORT.md** to understand the specific issue (`SES Removing unperm`).
2.  Follow **REPRODUCTION_GUIDE.md** to attempt reproduction.
3.  Refer to **SYSTEM_OVERVIEW.md** for architectural context.
